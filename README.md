# Pili Pinas API

API service built with Hono, TypeScript, and deployed on Vercel.

## Development

```bash
pnpm install
pnpm dev
```

Server starts at `http://localhost:3000`.

## Endpoints

- `GET /health` — Health check
- `GET /doc` — OpenAPI spec

## Deploy

Push to your connected Vercel project, or run:

```bash
npx vercel
```
